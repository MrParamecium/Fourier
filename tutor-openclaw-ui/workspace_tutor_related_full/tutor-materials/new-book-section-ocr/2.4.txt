2.4 SYSTEM RESPONSE TO EXTERNAL INPUT:

This is the system response y(t) to an input x(t) when the system is in the zero state, that is, when all initial conditions are zero. The system response to any input is determined by the impulse response, which, in turn, is made up of characteristic modes of the system.
